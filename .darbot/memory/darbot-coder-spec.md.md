- [FAILED] `:x:` - Failed/Error
- [WARNING] `:warning:` - Warning/Caution
- `:construction:` - Work in progress
- `:bulb:` - Tip/Idea
- [BUG] `:bug:` - Bug report
- `:sparkles:` - New feature
- [CRITICAL] `:fire:` - Critical/Hot
- [DOCS] `:memo:` - Documentation
- [UI] `:art:` - Design/UI
- `:zap:` - Performance
- `:lock:` - Security
- `:rocket:` - Deployment/Launch
